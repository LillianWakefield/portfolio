Chislango

August 2018

ChiSLANGo is a language learning game that displays Spanish slang from Mexico City alongside the literal English translation. The user selects from multiple choice options which is the actual meaning of the phrase. It was built using Node.js, PostgreSQL, Sequelize, and React.
